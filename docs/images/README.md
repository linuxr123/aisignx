# Documentation images

Place **screenshots and diagrams** here for the GitHub README and docs.

## Suggested files (contributions welcome)

| File | Shows |
|------|--------|
| `dashboard.png` | Main admin dashboard |
| `media-library.png` | Media list / upload |
| `playlist-editor.png` | Playlist detail with items |
| `display-player.png` | Browser or kiosk player |

Use **redacted** data (no real customer names, tokens, or private URLs).

## Using in README

After adding images, reference them from the root [README.md](../../README.md):

```markdown
![Dashboard](docs/images/dashboard.png)
```

The project logo lives at `server/static/img/AISignX.png` and is already linked from the README.
