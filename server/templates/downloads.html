Setup Instructions
  1. Install the client on your display device using one of the packages above.
  2. Launch the app. It will show a setup screen asking for your Server URL:
    {{ server_url }}
  3. Enter a friendly name for the display (e.g. Lobby Screen) and submit.
  4. Go to Displays — a pending registration will appear at the top. Click Approve and set the display name.
  5. The client detects approval automatically and starts playing your content.
Build from Source
Electron (Windows & Linux)
cd electron-client
npm install
npm run build:all
# Output: electron-client/dist/
# Copy to: static/clients/
Android
cd android-client
./gradlew assembleRelease
# APK: app/build/outputs/apk/release/
# Copy to: static/clients/AISignX-Player.apk

After copying packages to static/clients/, update the version numbers in the manifest editor above so clients know to download the new build.
